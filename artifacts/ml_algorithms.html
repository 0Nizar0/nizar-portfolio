Machine Learning Algorithm Types

01
🌿
Decision Tree
Supervised

Classifies data through a series of yes/no questions about input features, branching into narrower categories until a final prediction is reached. Fully interpretable — every decision can be traced.

Domains
Tabular Data
Use Cases
  • Loan approval decisions in banking
  • Medical diagnosis from patient symptoms
  • Customer churn prediction
02
🌲
Random Forest
Supervised

An ensemble of many Decision Trees each trained on a random data subset. The final prediction is a majority vote across all trees, making it significantly more robust and accurate than a single tree.

Domains
Tabular Data
Use Cases
  • Fraud detection in financial transactions
  • Stock market risk assessment
  • Disease prediction from clinical data
03
📈
Linear Regression
Supervised

Models the relationship between input features and a continuous output by fitting a straight line through the data, minimizing the error between predicted and actual values to forecast numerical outcomes.

Domains
Tabular Data
Use Cases
  • Real estate price prediction
  • Sales revenue forecasting
  • Energy consumption estimation
04
⚡
Support Vector Machine
Supervised

Finds the optimal boundary that best separates data into classes with the largest possible margin between them. Highly effective in high-dimensional spaces and where classification precision matters most.

Domains
Tabular Data Computer Vision
Use Cases
  • Spam email classification
  • Cancer cell detection from imaging
  • Text sentiment classification
05
🔵
K-Means Clustering
Unsupervised

Groups data points into K clusters by iteratively assigning each point to its nearest cluster center, then recalculating centers. Discovers natural groupings in data without needing any labels.

Domains
Tabular Data
Use Cases
  • Customer segmentation in marketing
  • Document topic grouping
  • Anomaly detection in network traffic
06
🔮
Principal Component Analysis
Unsupervised

Reduces dataset dimensionality by identifying the directions that capture the most variance. Simplifies complex, high-dimensional data while preserving its most meaningful structure and patterns.

Domains
Tabular Data Computer Vision
Use Cases
  • Feature reduction before model training
  • Face recognition preprocessing
  • Genomics and bioinformatics analysis
07
👁️
Convolutional Neural Network
Supervised

Uses convolutional layers to automatically detect spatial features like edges, textures, and shapes from images. Deeper layers progressively learn more abstract representations enabling powerful visual understanding.

Domains
Computer Vision
Use Cases
  • Medical image diagnosis (X-rays, MRIs)
  • Autonomous vehicle object detection
  • Facial recognition systems
08
🧠
Transformer (GPT / BERT)
Supervised

Uses self-attention mechanisms to capture relationships between all words in a sequence simultaneously. Transformers power most modern NLP systems, understanding long-range context and meaning across text.

Domains
NLP Generative AI
Use Cases
  • ChatGPT conversational AI
  • Google search query understanding
  • Automated document summarization
09
🎨
Generative Adversarial Network
Unsupervised

Two networks — a Generator and a Discriminator — compete with each other. The Generator creates synthetic data while the Discriminator tries to detect it as fake. Over time both improve, producing increasingly realistic outputs.

Domains
Computer Vision Generative AI
Use Cases
  • Synthetic image and video generation
  • Synthetic training data creation
  • Artistic style transfer
10
🌊
Diffusion Models
Unsupervised

Learns to generate content by training on gradually adding noise to data, then learning to reverse that process step by step. Currently powers leading image generation systems with highly realistic, detailed outputs.

Domains
Computer Vision Generative AI
Use Cases
  • DALL·E and Stable Diffusion image generation
  • Drug molecule design in pharma research
  • Video and audio content synthesis
